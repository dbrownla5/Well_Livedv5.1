# Listing & Photo Workflow — From Item to Listed in Minutes

**For:** Dayna Brown
**Goal:** Turn "I have an item to sell" into a live, on-brand listing in under
10 minutes — without paid photo apps and without rewriting copy from scratch.

This is a process, not another app. Keep it open while you list.

---

## Part 1 — Photos (5 minutes, free tools)

You don't need a subscription editor. Resale buyers decide on **clarity and
trust**, not artistry. The four edits that actually move items:

1. **Clean, neutral background.** Shoot against a single plain wall, a bedsheet,
   or a wood floor. One surface, no clutter. (Free: your phone's built-in
   "remove background" or the free tier of any editor — but a real plain
   backdrop beats any AI cutout.)
2. **True color.** Buyers return items that "looked different." Shoot in daylight
   near a window, no overhead yellow light. Tap to focus, then drag exposure so
   the color matches real life. Don't over-filter.
3. **Scale + fit.** One photo with a size reference (on a hanger, on a body, or
   next to a common object). This single photo cuts "what size is this" messages.
4. **The flaw shot.** Photograph any wear, mark, or repair up close. Counter-
   intuitive, but it *raises* trust and *lowers* returns and disputes.

**Shot list per item (always the same 5):**
- [ ] Front, full, clean background
- [ ] Back / reverse
- [ ] Scale or on-body
- [ ] Label / brand tag / authenticity mark
- [ ] Any flaw (or one detail shot if flawless)

---

## Part 2 — Description (3 minutes, your voice, not generic)

Use the skeleton below. It's built to match your brand voice (human-first,
business-always — never SEO-sanitized, never a feature menu). Fill the brackets.

```
[One true sentence about the piece — what it is and why it's worth a second look.]

[One sentence of honest context: condition, era, why you're moving it on.]

The details:
— [Brand / maker]
— [Size, with measurements: e.g. pit-to-pit, length]
— [Material / care]
— [Condition: name any flaw plainly]

[One closing line — not a pitch, a fact. What this piece is good for, who it's for.]
```

**Voice guardrails (the difference between your copy and everyone else's):**
- Lead with discovery, not specs. The buyer is uncovering something, not
  scanning a product sheet.
- Short punctuation lines for emphasis. Not for style.
- Never write a feature menu. Each detail is something they hadn't thought to
  look for.
- No SEO-speak, no "elevate your wardrobe" filler, no hype adjectives.

> **This repo already has a voice checker.** Once we re-lock your voice profile
> (step C), you can paste a draft description to `/api/voice/analyze` and it will
> flag exactly where the copy drifts off-brand before you post. The skeleton
> above is the manual version of the same standard.

---

## Part 3 — Post (2 minutes)

- [ ] Title: `[Brand] [Item] [Size] — [one distinguishing word]`
      (e.g. "Eileen Fisher silk shell — M — unworn")
- [ ] Price: anchor to comps, not hope. Check 2–3 sold (not listed) comparables.
- [ ] Pick the platform by tier (your agreement's tiers):
      - Standard / Contemporary → fast-moving marketplace
      - Designer / Luxury → authenticated platform (protects your split)
- [ ] Cross-post the same 5 photos + description everywhere at once.

---

## The repeatable loop (pin this)

1. Stage surface once. Photograph the **same 5 shots** for every item.
2. Fill the **same description skeleton**.
3. (After step C) Run the draft through the voice checker; fix flags.
4. Post with the tier→platform rule.
5. Log the item in the Handshake inventory so payout tracks automatically.

Same five steps every time. That's how listing goes from an hour to minutes —
the thinking is done once and lives here, not in your head each time.
